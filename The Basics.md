### 15. Initialization
    - git init "원하는 폴더이름" : 사용자가 정한 이름의 깃 리포지토리가 생성된다
      - $ git init demo : demo 폴더가 생성된다.
    - 디폴트 브랜치 master : 깃 리포지토리가 생성됬을 때는 디폴트로 마스터 브랜치로 설정된다.
